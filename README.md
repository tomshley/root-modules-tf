# Tomshley Root Modules TF

Terraform/OpenTofu root modules for Tomshley infrastructure provisioning.

This repository is part of the **Tomshley – OSS IP Division** and is maintained by **Tomshley LLC**.

---

## Overview

This repository contains root-level Terraform modules used to provision and manage Tomshley infrastructure. Modules are organized under the `terraform/` directory and are designed to be composed with the Breakground provisioning workflow.

---

## Usage

```bash
cd terraform/<module>
make init
make plan
make apply
```
